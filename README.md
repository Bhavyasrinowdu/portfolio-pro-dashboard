# PortfolioPro – Student Learning Dashboard

I built this project to solve a problem I personally faced while learning Git, GitHub, and preparing for hackathons. In the beginning, everything felt scattered — I didn’t know what to learn first or how to track my progress. So I decided to create a simple dashboard that organizes learning into small, clear steps.

## What this project is about

This is a basic web application that helps students follow a structured path while learning important tools like Git and GitHub. Instead of watching random tutorials, users can complete tasks one by one and actually track their progress.

## What you can do with it

- Go through three learning sections: Git, GitHub, and Hackathons  
- Complete small, practical tasks step by step  
- Mark tasks as done and see your progress increase  
- Continue from where you left off, even after refreshing  

## How I built it

I used simple web technologies because I wanted to focus more on logic and usability rather than complexity.

- HTML for structure  
- CSS for styling  
- JavaScript for functionality  
- LocalStorage to save progress  

## How it works (in simple terms)

When a user logs in, the dashboard opens with different sections. Each section contains tasks with short instructions. When a task is marked as completed, the progress bar updates automatically. The progress is saved in the browser, so it doesn’t reset every time.

## What I learned from this project

While building this, I understood:
- How to design a simple but useful user interface  
- How to manage data using JavaScript  
- How to think from a user’s perspective  
- How to break a big problem into smaller steps  

## Why I made this

I didn’t want other beginners to feel lost like I did. This project is my way of making the learning process a little more structured and less confusing.
